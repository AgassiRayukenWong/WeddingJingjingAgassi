项目名称 : The Last One 2D Remastered
平台 : Windows & Flash WebGL
年: 2020



尝试
下载 Windows 版

描述 :


TLO2D 是一款在线皇家战争游戏，其目标是成为战场上与其他敌人对抗的最后一名幸存者。


该游戏是由 EPITECH 的一组学生在两年的时间内开发的。 该项目很成功，即使计算机配置效率低下，也可以在 Windows 上通过 Installer 或 Flash WebGL 进行播放。
